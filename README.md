# lw-training-leo

## 1. Clone this project

```
https://github.com/Changhao029/randomcat.git
```
## 2. Make sure you have installed npm and node
```
brew install node
```
verify Node.js and NPM installation
```
node -v
```
```
npm -v
```

## 3. Install Vite
Need to install node.js 18+ in advance.
```
npm create vite@latest
```

## 4. Run the Project
```
npm install
npm run dev
```
